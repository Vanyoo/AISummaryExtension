# 更新日志 (CHANGELOG)

本文档记录了 AI Element Selector 扩展的所有版本更新和变更。

## [1.0.0] - 2026-01-15

### 🎉 首次正式发布

#### 核心功能
- **智能元素选择** - 点击网页任意元素，自动提取文本内容
- **AI 实时总结** - 支持流式输出，实时显示 AI 总结结果
- **Markdown 渲染** - 完整支持 Markdown 格式渲染（标题、列表、代码块、表格等）
- **Shadow DOM 支持** - 完美支持 Shadow DOM 内容的提取
- **拖拽调整大小** - 模态窗口支持拖拽移动和调整大小
- **思考过程展示** - 支持 `<thinking>` 标签，可查看 AI 思考过程

#### 技术特性
- **流式输出优化** - 使用 `requestAnimationFrame` 实现平滑渲染，无页面跳动
- **防抖机制** - 高效批量处理 DOM 更新，提升性能
- **内存安全** - 完善的资源清理机制，防止内存泄漏
- **防御性编程** - 全面的错误处理和降级方案
- **CORS 绕过** - 支持多种 API 调用模式（直连、代理、GM Bridge）

#### 用户体验
- **无蒙版设计** - 透明浮层，不影响原页面操作
- **快捷键支持** - ESC 键关闭模态窗口或退出选择模式
- **持久化配置** - 配置自动保存到 Chrome Storage
- **固定位置功能** - 可固定模态窗口位置
- **双击最大化** - 双击标题栏最大化/恢复窗口

#### 修复问题
- ✅ 修复 config 变量重复声明错误
- ✅ 修复 marked.js 加载失败问题
- ✅ 修复流式渲染时的页面跳动问题
- ✅ 统一日志输出系统

## 版本说明

### 版本号格式
遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/) 规范：

- **主版本号** (Major): 不兼容的 API 修改
- **次版本号** (Minor): 向下兼容的功能性新增
- **修订号** (Patch): 向下兼容的问题修正

### 变更类型标识
- 🎉 **新增** (Added): 新功能
- 🔧 **改进** (Changed): 现有功能的改进
- 🐛 **修复** (Fixed): Bug 修复
- 📝 **文档** (Documentation): 文档更新
- ♻️ **重构** (Refactor): 代码重构
- ⚡ **性能** (Performance): 性能优化
- 🔒 **安全** (Security): 安全修复
- 🗑️ **删除** (Removed): 功能删除
